`irxforge` is an R package for generating, parsing, and converting data
sets in pharmacometrics.

## Installation

Install the development version from
[GitHub](https://github.com/InsightRX/irxforge) with:

``` r
# install.packages("pak")
pak::pak("InsightRX/irxforge")
```

## Documentation

See at
[`https://insightrx.github.io/irxforge/`](https://insightrx.github.io/irxforge/reference/index.html)
and also in the installed package: `help(package = "irxforge")`.
